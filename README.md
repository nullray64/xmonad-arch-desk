# xmonad-arch-desk
Collection of configurations for setting up xmoad desktop from scratch

## Paths
`xmonad.hs & xmobarrc` at `$HOME/.xmonad/`

`alacritty.toml and aura-theme.toml` at `$HOME/.config/alacritty`

`wallp.png` at `$HOME/.wallp.png`

## Prerequisites
Using ArchLinux distibution:
- ttf-ubuntu-font-family
- xmonad
- xmobar
- betterlockscreen (AUR)
- rofi
- picom
- alacritty
